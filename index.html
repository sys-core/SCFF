Scff

Un script fuzzeur de fichier pour trouver des bogues dans les analyseurs de format de fichier.

View on GitHub Download .zip Download .tar.gz

Bienvenue à la page de SCFF.

Vulnérabilités de format de fichier sont en train de devenir le vecteur de choix pour les attaques côté client, si naturellement nous devrions être intéressés à trouver des bogues dans les analyseurs de format de fichier. Nous vous présentons un script qui vous permet de découvrir l'existence de bogues dans tout type d'analyseur de fichier.

Comment l'utiliser

Dans un prompt de commande lancez la commande comme dans l'exemple suivant:

python Fuzzeur_de_Fichier.py -e C:\WINDOWS\system32\notepad.exe -x .txt

-e: est le chemin vers l'exécutable de l'application cible. -x: est l'extension du fichier que vous voulez tester.
